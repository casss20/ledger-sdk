# Citadel Open Specification

> **Status: Not yet published.** This package is a placeholder for the planned open specification.

This directory will contain the public schemas, policy formats, token specifications (`gt_`),
and interface definitions for the Citadel ecosystem.

## Planned contents

- `schemas/` — JSON Schema definitions for actions, decisions, audit events
- `policies/` — Policy DSL grammar and examples
- `tokens/` — Governance token (`gt_`) format specification
- `openapi/` — OpenAPI 3.1 spec for the REST API
