Core Concepts

Understanding how Citadel works will help you design effective governance policies:

  • Policies — ALLOW, BLOCK, and APPROVAL_REQUIRED rules with precedence
  • Audit Trail — cryptographically chained, tamper-proof decision log
  • Human Approvals — pause execution and route to a human reviewer
  • Kill Switch — instantly block all agent actions globally or per-tenant
